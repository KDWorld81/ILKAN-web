## 👋 Commit Message 규칙

| **메시지 타입** | **설명** |
| --- | --- |
| **feat** | ✨ 새로운 기능 추가 및 기존 기능 수정 |
| **fix** | 🐛 버그 수정 |
| **docs** | 📚 문서 및 주석 수정 |
| **style** | 🎨 코드 스타일 및 포맷팅 수정 |
| **refact** | ♻️ 기능 변화 없는 코드 리팩터링 |
| **test** | ✅ 테스트 코드 추가/수정 |
| **chore** | 🔧 패키지 매니저 수정 및 기타 잡다한 변경(ex: `.gitignore`) |
| **merge** | 🔀 브랜치 병합 |

💡 **사용 예시:**  
feat: 로그인 폼 생성 완료
<br>
fix: 회원가입 에러 수정

<br>
